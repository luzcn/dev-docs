# Vim basic commands

#### Delete
- `daw` delete the current word.
- `dd` delete the entire line.
- `u` undo

#### copy/cut/paste
- `yy`  copy the current line, `p` to paste.
- `yw` copy to the next word,  `yiw`  copy the current word excluding the surrounded whitespaces, `yaw` includes spaces.
- `dw` to cut the word.
